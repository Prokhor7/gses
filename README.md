# gses

## Prerequisites

Ensure you have the following installed on your system:

- Docker
- Docker Compose

## Setup

1. **Clone the repository:**

```bash
git clone https://github.com/Prokhor7/gses.git
cd gses
```
2. **Build and run**

```bash
docker compose up --build
```

2. **To stop**

```bash
docker compose down
```
